# RAENNE
RAÉNNE

Founded by two visionary teenage girls, RAÉNNE is a high-end eCommerce fashion label that redefines luxury through the lens of youth, confidence, and modern femininity. Selling clothing and accessories as the main products.

Design Process
We created this website targeted towards the woman of all ages. Those who value aesthetics, convinence, and brand identity. Particularlly, this website are for the digitally native shoppers who enjoy interactive and rewarding experciences

External source code used: w3schools, Chatgpt

- As a fashion shopper, I want to browse curated collections, so that I can discover clothing and accessories that match my style.
- As a first-time user, I want to sign up for an account, so that I can access personalised features and rewards.
- As a user, I want to participate in events and spin-the-wheel rewards, so that shopping feels fun and engaging.
- As a user, I want to add items to my shopping bag, so that I can review my choices before purchasing.
- As a user, I want a simple checkout process, so that I can complete my purchase quickly and smoothly.

Features


Existing Features
Feature 1 - Choose Location
Users can select their location for easy payment

Feature 2 - Create an Account
Allows users to register an account by entering their details

Feature 3 - Login to the Account
Registered users to log in securely

Feature 4 - Navigation Bar
An intuitive menu, header, and footer that help users find their way around the site. 

Feature 5 - Contact Us
Submit enquiries or feedback so that they can communicate directly with the brand.

Feature 6 - 3d Models
Allows users to view products in 3D so that they can better visualise clothing and accessories from multiple angles.

Feature 7 - Carousel for Individual Clothes
Users can scroll through multiple images of a product so that they can view details and styling options.

Feature 8 - Spin the Wheel
Participate in a gamified reward feature so that they can earn discounts, points, or exclusive rewards.

Feature 9 - Shopping Bag
Allows users to add, review, and remove items before checkout so that they can manage their purchases easily.

Feature 10 - Input Credit Card/Email
Users to input payment and contact details securely so that they can complete the checkout process.

Feature 11 - Lottie
Uses Lottie animations to provide visual feedback and enhance user engagement during interactions.

Feature 12 - Mobile Responsiveness
A design that adapts to and looks good on all devices, including desktops, tablets, and phones.
